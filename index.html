Introduction

Item Name : Paribahan ticket search, booking and purchased system

Item Version : v 1.0

Author : Paribahan Devs

First of all, Thank you so much for using our API.

This documentation is to help you regarding each step of customization. Please go through the documentation carefully to understand how this Works.

API documentaion are arranged in order by which they will need to be called.

Important notes

  1. 1. First you will need username and password to get API data
  2. 2. You will have to provied user access token every time you call API
  3. 3. Only those bus will be shown for a user which are allowed to a user.
  4. 4. When a tickets are booked then it will available for a user for certain amount of time to complete the process.
  5. 5. Not every time or every user or every bus ticket will have the access to cancel ticket.
  6. 6. Authority will preserve the right to do anything

Getting Started

Step 01) Get Access token

Url : base_url/login
Method/Action Type : Form POST
Return Response : json format
NOTE :

Body / Params

Param name Type Value Required
username String your secret user name Yes
password String your secret user name Yes

Sample Response

Get trips

Step 03) Get trips

Url : base_url/trip/search-trips?from=Dhaka&to=Cox's Bazar&busType=ALL&date=2023-03-03
Method/Action Type : Form GET
Return Response : json format
NOTE :

Body / Params

Param name Type Value Required
from String City name from the city list Yes
to String City name from the city list Yes
busType String export enum BusType { AC = 'AC', NON_AC = 'NON_AC', ALL = 'ALL', } Yes
date String Date must be greater than today Yes

Sample Response

                                        
                                            {
                                                "from": "Dhaka",
                                                "to": "Bandarban",
                                                "BusType": "",
                                                "Date": "05-03-2023"
                                            }
